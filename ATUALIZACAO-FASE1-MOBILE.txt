HYDRA WALLET — FASE 1 MOBILE E PROTEÇÃO DA API

Incluído:
- Cooldown persistente de 60 segundos por usuário após chave inválida ou falha confirmada.
- Pix Copia e Cola em texto puro + arquivo .txt.
- Menu de mensagem "Pagar esta chave": no celular, segure a mensagem > Apps > Pagar esta chave.
- O modal pede somente o valor e reutiliza o fluxo seguro Sim/Não já existente.
- Menu de mensagem "Copiar chave Pix": devolve a chave em texto puro + arquivo .txt.

Segurança:
- Nenhuma tabela de saldo foi removida ou recriada.
- O arquivo hydra-wallet.db não deve ser substituído na VPS.
- Faça backup antes da atualização.

Após subir os arquivos:
1. npm install
2. npm run deploy
3. pm2 restart hydra-wallet --update-env
